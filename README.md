# Quick Start
1) Clone the repository:
```bash
git clone https://github.com/GriffithS17/Using-a-Public-API.git
```
2) Navigate to the Using-a-Public-API folder:
```bash
cd Using-a-Public-API/
```
3) Install the dependencies:
```bash
npm i
```
4) Make sure Node.js is installed on your system.

5) Run `index.js` with Node.js:
```bash
node index.js
```
6) Open http://localhost:3000/ in your browser.
