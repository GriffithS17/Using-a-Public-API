# Quick Start
1) Make sure Node.js installed on your system.

2) Clone the repository:
```bash
git clone https://github.com/GriffithS17/Using-a-Public-API.git
```
3) Navigate to the Using-a-Public-API folder:
```bash
cd Using-a-Public-API/
```
4) Install the dependencies:
```bash
npm i
```
5) Run `index.js` with Node.js:
```bash
node index.js
```
6) Open http://localhost:3000/ in your browser.
